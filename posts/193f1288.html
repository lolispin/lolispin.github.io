2. 数学原理

我们探究下上述的分解在何种情况下能够不断进行，并且有效地控制误差。从 pip_ipi​ 和 qiq_iqi​ 的形式上来看，首先分母必定不为 000，也就是

bi+aipi−1≠0b_i + a_i p_{i-1} \neq 0 bi​+ai​pi−1​=0

此外，为了是误差被控制，必须有

∣pi∣<1|p_i| < 1 ∣pi​∣<1

现在给出如下的定理：如果系数矩阵满足行对角占优，即

∣bi∣>∣ai∣+∣ci∣|b_i| > |a_i| + |c_i| ∣bi​∣>∣ai​∣+∣ci​∣

那么上面两个条件将同时满足。

考虑绝对值不等式

∣bi+aipi−1∣>∣bi∣−∣ai∣∣pi∣|b_i + a_i p_{i-1}| > |b_i| - |a_i||p_i| ∣bi​+ai​pi−1​∣>∣bi​∣−∣ai​∣∣pi​∣

因此我们只需要证明 ∣pi∣<1|p_i| < 1∣pi​∣<1 即可。考虑数学归纳法，当 i=1i = 1i=1 时，有

p1=−c1b1  ⟹  ∣p1∣=∣c1∣∣b1∣<1p_1 = - \frac{c_1}{b_1} \implies |p_1| = \frac{|c_1|}{|b_1|} < 1 p1​=−b1​c1​​⟹∣p1​∣=∣b1​∣∣c1​∣​<1

成立。现在假定第 i−1i - 1i−1 步成立，那么第 iii 步为

∣pi∣=∣ci∣∣bi+aipi−1∣<∣ci∣∣bi∣−∣aipi−1∣<∣ci∣∣bi∣−∣ai∣<∣ci∣∣ci∣=1|p_i| = \frac{|c_i|}{|b_i + a_i p_{i-1}|} < \frac{|c_i|}{|b_i| - |a_i p_{i-1}|} < \frac{|c_i|}{|b_i| - |a_i|} < \frac{|c_i|}{|c_i|} = 1 ∣pi​∣=∣bi​+ai​pi−1​∣∣ci​∣​<∣bi​∣−∣ai​pi−1​∣∣ci​∣​<∣bi​∣−∣ai​∣∣ci​∣​<∣ci​∣∣ci​∣​=1

也成立，因此对于任意的正整数 iii，始终有 ∣pi∣<1|p_i| < 1∣pi​∣<1 成立。

另一方面，如果是列对角占优，也就是

∣bi∣>∣ai+1∣+∣ci−1∣|b_i| > |a_{i + 1}| + |c_{i - 1}| ∣bi​∣>∣ai+1​∣+∣ci−1​∣

那么进行自下而上的消元，也就是

可以解出

Pi=−aibi+ciPi+1P_i = \frac{-a_i}{b_i + c_i P_{i+1}} Pi​=bi​+ci​Pi+1​−ai​​

Qi=fi−ciQi+1bi+ciPi+1Q_i = \frac{f_i - c_i Q_{i+1}}{b_i + c_i P_{i+1}} Qi​=bi​+ci​Pi+1​fi​−ci​Qi+1​​

可以证明，此时列对角占优满足分母不为 000 以及控制误差的要求。

3. 代码实现

这里我只给出主对角占优的代码。用户需要输入四个矩阵，而不单单是一个系数矩阵和一个常数矩阵。此外，由于求取 xix_ixi​ 需要知晓 pip_ipi​、qiq_iqi​ 和 xi+1x_{i + 1}xi+1​ 这三个量，因此利用一个循环结构求出 pip_ipi​ 和 qiq_iqi​ 后，无法在循环中同时求出 xix_ixi​。

在循环中，我先计算了重复的部分 denominator = 1/(B(i) + A(i)*p(i - 1));，不过这里英文名称和其内涵有所错位，原因在于一开始我写的是 denominator = B(i) + A(i)*p(i - 1);，确实是直接计算分母。然后后续在计算 pip_ipi​ 和 qiq_iqi​ 时，需要两次对其做除法，消耗了资源。
