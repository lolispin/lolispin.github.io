最速下降法

1. 问题

在实际问题中，通常系数矩阵会是对称正定矩阵。对于这种特殊的矩阵，有着特别的方法。

2. 数学原理

先让我们看看什么是正定矩阵吧。方阵 A∈Rn×nA \in \mathbb{R}^{n \times n}A∈Rn×n，其对于任意向量 x∈Rnx \in \mathbb{R}^{n}x∈Rn，都有

xTAx≥0x^T A x \ge 0 xTAx≥0

则称 AAA 为半正定矩阵。若大于号严格成立，则为正定矩阵。注意，正定矩阵不一定对称。若 AAA 为对称正定矩阵，那么起将具有如下的性质

  • AAA 非奇异，且 A−1A^{-1}A−1 也对称正定

  • AAA 的所有特征值都是正实数

  • AAA 的所有顺序主子阵都对称正定的

  • AAA 的所有顺序主子式都大于零

现在，考虑如下的定理：

设 A∈Rn×nA \in \mathbb{R}^{n \times n}A∈Rn×n 对称正定，则 x∗x_*x∗​ 是 Ax=bAx = bAx=b 的解当且仅当 x∗x_*x∗​ 是最小化问题

min⁡x∈RnΦ(x)≜12xTAx−bTx\min_{x \in \mathbb{R}^n} \Phi(x) \triangleq \frac{1}{2}x^T A x - b^T x x∈Rnmin​Φ(x)≜21​xTAx−bTx

的解，即 Φ(x)\Phi(x)Φ(x) 的最小值点。

于是，我们将一开始的求解线性方程组的问题转换为了求最值，或者极值的问题

Ax=b  ⟺  min⁡Φ(x)Ax = b \iff \min\Phi(x) Ax=b⟺minΦ(x)

求解最小化问题 min⁡x∈RnΦ(x)\displaystyle\min_{x \in \mathbb{R}^n} \Phi(x)x∈Rnmin​Φ(x) 的一类常用方法是线搜索方法：

给定初始值 x(0)x^{(0)}x(0)，确定一个使得 Φ(x)\Phi(x)Φ(x) 变小的方向 p1∈Rnp_1 \in \mathbb{R}^np1​∈Rn，即下降方向，满足 ∥p1∥=1\|p_1\| = 1∥p1​∥=1，然后计算步长 α1∈R+\alpha_1 \in \mathbb{R}_+α1​∈R+​，使得 Φ(x)\Phi(x)Φ(x) 沿该下降方向达到最小，即

x(1)=x(0)+α1p1x^{(1)} = x^{(0)} + \alpha_1 p_1 x(1)=x(0)+α1​p1​

其中

α1=arg⁡min⁡α>0Φ(x(0)+αp1)\alpha_1 = \arg\min_{\alpha > 0} \Phi\left(x^{(0)} + \alpha p_1\right) α1​=argα>0min​Φ(x(0)+αp1​)

那么现在有两个问题：

  • 如何确定 ppp
  • 如何确定 α\alphaα

关于 ppp 的确定，我们希望的是

Φ(x(0)+αp1)−Φ(x(0))<0\Phi(x^{(0)} + \alpha p_1) - \Phi(x^{(0)}) < 0 Φ(x(0)+αp1​)−Φ(x(0))<0

因此，考虑多元函数的 Taylor 展开公式

Φ(x)=Φ(x(0))+(x−x(0))T∇Φ(x(0))+o(∥x−x(0)∥)\Phi(x) = \Phi(x^{(0)}) + (x - x^{(0)})^T \nabla \Phi(x^{(0)}) + o(\|x - x^{(0)}\|) Φ(x)=Φ(x(0))+(x−x(0))T∇Φ(x(0))+o(∥x−x(0)∥)

记 ppp 为 x−x(0)x - x^{(0)}x−x(0) 所在的方向，即

p=x−x(0)∥x−x(0)∥p = \frac{x - x^{(0)}}{\|x - x^{(0)}\|} p=∥x−x(0)∥x−x(0)​

则

(x−x(0))T∇Φ(x(0))=∥x−x(0)∥⋅pT∇Φ(x(0))(x - x^{(0)})^T \nabla \Phi(x^{(0)}) = \|x - x^{(0)}\| \cdot p^T \nabla \Phi(x^{(0)}) (x−x(0))T∇Φ(x(0))=∥x−x(0)∥⋅pT∇Φ(x(0))

于是得到

Φ(x)−Φ(x(0))=∥x−x(0)∥⋅pT∇Φ(x(0))\Phi(x) - \Phi(x^{(0)}) = \|x - x^{(0)}\| \cdot p^T \nabla \Phi(x^{(0)}) Φ(x)−Φ(x(0))=∥x−x(0)∥⋅pT∇Φ(x(0))

因此，只要满足 pT∇Φ(x(0))<0p^T \nabla \Phi(x^{(0)}) < 0pT∇Φ(x(0))<0，则 ppp 就是下降方向，下降速度最快的方向为 pT∇Φ(x(0))p^T \nabla \Phi(x^{(0)})pT∇Φ(x(0)) 最小。根据 Cauchy-Schwarz 不等式可知

∣pT∇Φ(x(0))∣≤∥p∥⋅∥∇Φ(x(0))∥,|p^T \nabla \Phi(x^{(0)})| \leq \|p\| \cdot \|\nabla \Phi(x^{(0)})\|, ∣pT∇Φ(x(0))∣≤∥p∥⋅∥∇Φ(x(0))∥,

等号当且仅当 ppp 与 ∇Φ(x(0))\nabla \Phi(x^{(0)})∇Φ(x(0)) 共线时成立。因此当

p1=−∇Φ(x(0))∥∇Φ(x(0))∥p_1 = -\frac{\nabla \Phi(x^{(0)})}{\|\nabla \Phi(x^{(0)})\|} p1​=−∥∇Φ(x(0))∥∇Φ(x(0))​

时，pT∇Φ(x(0))p^T \nabla \Phi(x^{(0)})pT∇Φ(x(0)) 达到最小。我们称该下降方向为最速下降方向，相应的线搜索方法为最速下降法。由于最速下降方向就是 Φ(x)\Phi(x)Φ(x) 在当前迭代点处的负梯度方向，因此也称为负梯度方向法。

由于 AAA 对称正定，直接计算可得

Φ(x(0)+αp1)=12(x(0)+αp1)TA(x(0)+αp1)−bT(x(0)+αp1)=12α2p1TAp1+αp1TAx(0)+12(x(0))TAx(0)−bTx(0)−αbTp1=12α2p1TAp1−αp1T(b−Ax(0))+Φ(x(0))=12α2p1TAp1−αp1Tr0+Φ(x(0))\begin{aligned} \Phi(x^{(0)} + \alpha p_1) &= \frac{1}{2} (x^{(0)} + \alpha p_1)^T A (x^{(0)} + \alpha p_1) - b^T (x^{(0)} + \alpha p_1) \\ &= \frac{1}{2} \alpha^2 p_1^T A p_1 + \alpha p_1^T A x^{(0)} + \frac{1}{2} (x^{(0)})^T A x^{(0)} - b^T x^{(0)} - \alpha b^T p_1 \\ &= \frac{1}{2} \alpha^2 p_1^T A p_1 - \alpha p_1^T (b - A x^{(0)}) + \Phi(x^{(0)}) \\ &= \frac{1}{2} \alpha^2 p_1^T A p_1 - \alpha p_1^T r_0 + \Phi(x^{(0)}) \end{aligned} Φ(x(0)+αp1​)​=21​(x(0)+αp1​)TA(x(0)+αp1​)−bT(x(0)+αp1​)=21​α2p1T​Ap1​+αp1T​Ax(0)+21​(x(0))TAx(0)−bTx(0)−αbTp1​=21​α2p1T​Ap1​−αp1T​(b−Ax(0))+Φ(x(0))=21​α2p1T​Ap1​−αp1T​r0​+Φ(x(0))​

其中 r0=b−Ax(0)r_0 = b - A x^{(0)}r0​=b−Ax(0)。这是关于 α\alphaα 的一元二次函数，且二次项系数为正，所以对其求导即可

df(α)dα=(p1TAp1)α−p1Tr0\frac{\mathrm{d} f(\alpha)}{\mathrm{d} \alpha} = (p_1^T A p_1) \alpha - p_1^T r_0 dαdf(α)​=(p1T​Ap1​)α−p1T​r0​

3. 代码实现

在前面，我们已经得到了 ppp 的表达式，并据此求出了 α\alphaα。不过问题来了，这里涉及到求梯度的问题。好在并不需要吧这部分交给计算机，而是我们可以先进行计算

∇Φ(x)=∇(12xTAx−bTx)=Ax−b\nabla \Phi(x) = \nabla \left( \frac{1}{2}x^T A x - b^T x \right) = Ax - b ∇Φ(x)=∇(21​xTAx−bTx)=Ax−b

因此

p1=Ax(0)−b=−r0p_1 = Ax^{(0)} - b = -r_0 p1​=Ax(0)−b=−r0​

当然了，有的朋友可能会问，现在我们仅仅求出了第一次循环中的 p1p_1p1​ 和 α1\alpha_1α1​。不过没关系，只要我们将第一次求得的 x(1)x^{(1)}x(1) 视为 x(0)x^{(0)}x(0) 而保持函数 Φ(x)\Phi(x)Φ(x) 不变继续迭代即可。具体而言

x(k)=x(k−1)+αkpkx^{(k)} = x^{(k-1)} + \alpha_k p_k x(k)=x(k−1)+αk​pk​

其中

pk=−∇Φ(x(k−1))=b−Ax(k−1)=rk−1p_k = -\nabla \Phi(x^{(k-1)}) = b - Ax^{(k-1)} = r_{k - 1} pk​=−∇Φ(x(k−1))=b−Ax(k−1)=rk−1​

αk=pkTrk−1pkTApk=rk−1Trk−1rk−1TArk−1\alpha_k = \frac{p_k^T r_{k-1}}{p_k^T A p_k} = \frac{r_{k-1}^T r_{k-1}}{r_{k-1}^T A r_{k-1}} αk​=pkT​Apk​pkT​rk−1​​=rk−1T​Ark−1​rk−1T​rk−1​​

最后来看一下收敛性，有如下定理：

设 A∈Rn×nA \in \mathbb{R}^{n \times n}A∈Rn×n 对称正定，则对任意初值 x(0)x^{(0)}x(0)，最速下降法都收敛，且

∥x(k+1)−x∗∥A∥x(k)−x∗∥A≤κ−1κ+1\frac{\|x^{(k+1)} - x_*\|_A}{\|x^{(k)} - x_*\|_A} \leq \frac{\kappa - 1}{\kappa + 1} ∥x(k)−x∗​∥A​∥x(k+1)−x∗​∥A​​≤κ+1κ−1​

其中 κ\kappaκ 为 AAA 的谱条件数，范数定义为

∥x∥A≜xTAx\|x\|_A \triangleq \sqrt{x^T A x} ∥x∥A​≜xTAx​

由步长的选取方法可知，最速下降法的迭代解具有下面的最优性

x(k)=arg⁡min⁡x∈x(k−1)+span{pk}Φ(x)x^{(k)} = \arg\min_{x \in x^{(k-1)} + \text{span}\{p_k\}} \Phi(x) x(k)=argx∈x(k−1)+span{pk​}min​Φ(x)

由于舍弃了 Taylor 展开式中的高阶项，因此最速下降方向是局部最优的。

此外，关于输入的系数矩阵对称正定性的判断，我计划对前者使用 A−ATA - A^TA−AT 的无穷范数来处理，后者则考察能否进行 Cholesky 分解。
