共轭梯度法

1. 问题

上一篇博客中我们看到，最速下降法的迭代次数要比 Gauss Seidel 迭代法多。此外，其还具有一些别的缺点。至于具体的缺点，目前我还未能找到好的参考资料加以描述，因此先搁置着。为了改善最速下降法的收敛性质，需要对下降方向进行改进，由此，共轭方向法应运而生。我们考虑取一组完备正交基

{p1, p2, …, pn}\{ p_1, \, p_2, \, \ldots, \, p_n \} {p1​,p2​,…,pn​}

据此进行展开

x∗−x(0)=α1p1+α2p2+⋯+αnpnx^* - x^{(0)} = \alpha_1 p_1 + \alpha_2 p_2 + \cdots + \alpha_n p_n x∗−x(0)=α1​p1​+α2​p2​+⋯+αn​pn​

其中系数为

αi=piT(x∗−x(0))piTpi\alpha_i = \frac{p_i^T(x^* - x^{(0)})}{p_i^Tp_i} αi​=piT​pi​piT​(x∗−x(0))​

但是这样会面临一个无法解决的问题：α\alphaα 的表达式中含有待求的真实解 x∗x_*x∗​。不过 Ax∗=bAx_* = bAx∗​=b 中常数项 bbb 是已知的，如果能够将 x∗x_*x∗​ 更换为 bbb，那就好了。

2. 数学原理

设 A∈Rn×nA \in \mathbb{R}^{n \times n}A∈Rn×n 对称正定，若非零向量 x,y∈Rnx, y \in \mathbb{R}^nx,y∈Rn 满足

yTAx=0y^T A x = 0 yTAx=0

则称 xxx 和 yyy 是 AAA-共轭的，也称 AAA-正交的。

此外，我们证明，对于 nnn 个相互 AAA-共轭的向量，一定是线性无关的。首先在这样的向量组中，向量之间不存在简单的倍数关系，这一点可由 AAA 的正定性保证

pTAp≥0p^TAp \ge 0 pTAp≥0

当且仅当 p=0p = 0p=0 时取等。那么现在考虑一组系数 {cn}\{ c_n \}{cn​}使得

c1p1+c2p2+⋯+ckpk=0c_1 p_1 + c_2 p_2 + \dots + c_k p_k = 0 c1​p1​+c2​p2​+⋯+ck​pk​=0

等式两边同时左乘 pjTAp_j^T ApjT​A，有

pjTA(c1p1+c2p2+⋯+ckpk)=pjTA⋅0p_j^T A (c_1 p_1 + c_2 p_2 + \dots + c_k p_k) = p_j^T A \cdot 0 pjT​A(c1​p1​+c2​p2​+⋯+ck​pk​)=pjT​A⋅0

具体展开可得

cj(pjTApj)=0c_j (p_j^T A p_j) = 0 cj​(pjT​Apj​)=0

显然有cj=0c_j= 0cj​=0。

那么现在可以使用一组互相共轭的向量组进行展开，向量的数量恰为 xxx 的维度。此时需要在两边分别左乘 pkTAp_k^T ApkT​A，可得

αk=pkTA(x∗−x(0))pkTApk=pkT(b−Ax(0))pkTApk\alpha_k = \frac{p_k^T A (x^* - x^{(0)})}{p_k^T A p_k} = \frac{p_k^T (b - Ax^{(0)})}{p_k^T A p_k} αk​=pkT​Apk​pkT​A(x∗−x(0))​=pkT​Apk​pkT​(b−Ax(0))​

这意味着可以通过下降方向 p1,p2,…,pnp_1, p_2, \dots, p_np1​,p2​,…,pn​ 和右端项 bbb 以及 x(0)x^{(0)}x(0) 将方程组的解表示出来。

相比于最速下降法的局部最优性，共轭方向法具有全局最优性：

x(k)=arg⁡min⁡x∈x(0)+span{p1,p2,…,pk}Φ(x)x^{(k)} = \arg\min_{x \in x^{(0)} + \text{span}\{p_1, p_2, \dots, p_k\}} \Phi(x) x(k)=argx∈x(0)+span{p1​,p2​,…,pk​}min​Φ(x)

以及有限步终止性质：若不考虑舍入误差，则至多 nnn 次迭代后，x(k)x^{(k)}x(k) 就是精确解。最后，我们给出一些能够简化计算的定理

设 A∈Rn×nA \in \mathbb{R}^{n \times n}A∈Rn×n 对称正定，共轭梯度法迭代 mmm 步后 (m<nm < nm<n)，对任意 kkk (k≤mk \leq mk≤m) 有

rkTri=0,i=0,1,2,…,k−1r_k^T r_i = 0, \quad i = 0, 1, 2, \dots, k - 1 rkT​ri​=0,i=0,1,2,…,k−1

rkTpi=0,i=1,2,…,kr_k^T p_i = 0, \quad i = 1, 2, \dots, k rkT​pi​=0,i=1,2,…,k

span{r0,r1,…,rk}=span{p1,p2,…,pk+1}=span{r0,Ar0,…,Akr0}\text{span}\{r_0, r_1, \dots, r_k\} = \text{span}\{p_1, p_2, \dots, p_{k+1}\} = \text{span}\{r_0, Ar_0, \dots, A^k r_0\} span{r0​,r1​,…,rk​}=span{p1​,p2​,…,pk+1​}=span{r0​,Ar0​,…,Akr0​}

该算法的收敛性由如下定理保证：

设 A∈Rn×nA \in \mathbb{R}^{n \times n}A∈Rn×n 对称正定，则对任意初值 x(0)x^{(0)}x(0)，共轭梯度法都收敛，且

∥x(k)−x∗∥A∥x(0)−x∗∥A≤2(κ−1κ+1)k\frac{\|x^{(k)} - x_*\|_A}{\|x^{(0)} - x_*\|_A} \leq 2 \left( \frac{\sqrt{\kappa} - 1}{\sqrt{\kappa} + 1} \right)^k ∥x(0)−x∗​∥A​∥x(k)−x∗​∥A​​≤2(κ​+1κ​−1​)k

其中 κ\kappaκ 为 AAA 的谱条件数，范数 ∥x∥A≜xTAx\|x\|_A \triangleq \sqrt{x^T A x}∥x∥A​≜xTAx​。

3. 代码实现

那么现在的问题在于，如何求出，或者选出这样的一组相互共轭的向量。理论上来说，可以选取任意的初向量。不过我们知道，最好的优化方向自然是填补上残差

r0=b−Ax0r_0 = b - Ax_0 r0​=b−Ax0​

因此可以将它选为 p1p_1p1​，由此可求出 α1\alpha_1α1​。为了求出 p2p_2p2​，考虑迭代解

x1=x0+α1p1x_1 = x_0 + \alpha_1 p_1 x1​=x0​+α1​p1​

然后求出 x1x_1x1​ 处的残差

r1=b−Ax1r_1 = b - Ax_1 r1​=b−Ax1​

如果选取 r1r_1r1​ 作为 p2p_2p2​，那这就是最速下降法，毫无新意。于是考虑选择

p2=r1+β1p1p_2 = r_1 + \beta_1 p_1 p2​=r1​+β1​p1​

并且我们要求其与 p1p_1p1​ 是 AAA-共轭的，于是

(r1+β1p1)TAp1=0(r_1 + \beta_1 p_1)^T A p_1 = 0 (r1​+β1​p1​)TAp1​=0

可解出

β1=−r1TAp1p1TAp1\beta_1 = - \frac{r_1^T A p_1}{p_1^T A p_1} β1​=−p1T​Ap1​r1T​Ap1​​

考虑到算法的实用性，当 nnn 很大时，通常不需计算所有的 αk\alpha_kαk​，而只要计算前面部分，得到一个满足精度要求的近似解即可。因此，我们把 x∗=x(0)+α1p1+⋯x_* = x^{(0)} + \alpha_1 p_1 + \cdotsx∗​=x(0)+α1​p1​+⋯ 看作是一个迭代过程：

x(k)=x(k−1)+αkpkx^{(k)} = x^{(k-1)} + \alpha_k p_k x(k)=x(k−1)+αk​pk​

改写 αk\alpha_kαk​ 的计算公式：由于

x(k−1)−x(0)=α1p1+α2p2+⋯+αk−1pk−1x^{(k-1)} - x^{(0)} = \alpha_1 p_1 + \alpha_2 p_2 + \cdots + \alpha_{k-1} p_{k-1} x(k−1)−x(0)=α1​p1​+α2​p2​+⋯+αk−1​pk−1​

所以

pkTA(x(k−1)−x(0))=0p_k^T A (x^{(k-1)} - x^{(0)}) = 0 pkT​A(x(k−1)−x(0))=0

即

pkTAx(0)=pkTAx(k−1)p_k^T A x^{(0)} = p_k^T A x^{(k-1)} pkT​Ax(0)=pkT​Ax(k−1)

因此

αk=pkT(b−Ax(0))pkTApk=pkT(b−Ax(k−1))pkTApk=pkTrk−1pkTApk\alpha_k = \frac{p_k^T (b - A x^{(0)})}{p_k^T A p_k} = \frac{p_k^T (b - A x^{(k-1)})}{p_k^T A p_k} = \frac{p_k^T r_{k-1}}{p_k^T A p_k} αk​=pkT​Apk​pkT​(b−Ax(0))​=pkT​Apk​pkT​(b−Ax(k−1))​=pkT​Apk​pkT​rk−1​​

于是我们有了

p1=r0=b−Ax0p_1 = r_0 = b - Ax_0 p1​=r0​=b−Ax0​

rk=b−Axkr_k = b - Ax_k rk​=b−Axk​

pk+1=rk+βkpkp_{k+1} = r_k + \beta_k p_k pk+1​=rk​+βk​pk​

βk=−rkTApkpkTApk\beta_k = -\frac{r_k^T A p_k}{p_k^T A p_k} βk​=−pkT​Apk​rkT​Apk​​

αk=pkTrk−1pkTApk\alpha_k = \frac{p_k^T r_{k-1}}{p_k^T A p_k} αk​=pkT​Apk​pkT​rk−1​​

此外，根据迭代公式可得

rk=b−A(xk−1+αkpk)=b−Axk−1−Aαkpk=rk−1−αkApkr_k = b - A(x_{k - 1} + \alpha_{k}p_{k}) = b - Ax_{k - 1} - A\alpha_{k}p_{k} = r_{k - 1} - \alpha_{k}Ap_{k} rk​=b−A(xk−1​+αk​pk​)=b−Axk−1​−Aαk​pk​=rk−1​−αk​Apk​

因此可以反复利用 ApkAp_kApk​ 来节省计算。还没完，利用前面的正交性定理，可以得到

αk=pkTrk−1pkTApk=(rk−1+βkpk−1)Trk−1pkTApk=rk−1Trk−1pkTApk\alpha_k = \frac{p_k^T r_{k-1}}{p_k^T A p_k} = \frac{(r_{k - 1} + \beta_kp_{k - 1})^T r_{k - 1}}{p_k^T A p_k} = \frac{r_{k - 1}^T r_{k-1}}{p_k^T A p_k} αk​=pkT​Apk​pkT​rk−1​​=pkT​Apk​(rk−1​+βk​pk−1​)Trk−1​​=pkT​Apk​rk−1T​rk−1​​

βk=−rkTApkpkTApk=−rkTpkTApkrk−1−rkαk=rkTrkrk−1Trk−1\beta_k = -\frac{r_k^T A p_k}{p_k^T A p_k} = -\frac{r_k^T}{p_k^T A p_k} \frac{r_{k - 1} - r_{k}}{\alpha_k} = \frac{r_k^T r_k}{r_{k - 1}^T r_{k-1}} βk​=−pkT​Apk​rkT​Apk​​=−pkT​Apk​rkT​​αk​rk−1​−rk​​=rk−1T​rk−1​rkT​rk​​
