Jacobi 迭代

1. 问题

在前面，我们已经得到了 Gauss 消去法、LU 分解法和 Cholesky 分解法，以及针对三对角矩阵的 Thomas 追赶法。然而，从复杂度上看，前三者都是 O(n3)O(n^3)O(n3) 的。对于大规模线性方程组，特别是稀疏方程组，当前的首选方法是迭代方法。考虑到计算机的浮点数计算，因此不必担心迭代法在精度上比前面精确的方法差。其基本思想为：对于

Ax=b,A∈Rn×n 非奇异Ax = b, \quad A \in \mathbb{R}^{n \times n} \text{ 非奇异} Ax=b,A∈Rn×n 非奇异

给定一个初始值 x(0)x^{(0)}x(0)，通过一定的迭代格式生成一个迭代序列 {x(k)}k=0∞\{x^{(k)}\}_{k=0}^\infty{x(k)}k=0∞​，使得

lim⁡k→∞x(k)=x∗≜A−1b\lim_{k \to \infty} x^{(k)} = x_* \triangleq A^{-1}b k→∞lim​x(k)=x∗​≜A−1b

迭代法分为定常迭代和子空间迭代法，目前我们介绍的 Jacobi 迭代、Gauss-Seidel 迭代和 SOR 都属于前者。其精神为采用某个矩阵 GGG 在一定程度上代替 AAA，记 Gx=bGx = bGx=b 的解为 x(1)x^{(1)}x(1)，与原方程的解 x∗=A−1bx_* = A^{-1}bx∗​=A−1b 之间的误差满足

A(x∗−x(1))=b−Ax(1).A \left( x_* - x^{(1)} \right) = b - Ax^{(1)}. A(x∗​−x(1))=b−Ax(1).

如果 x(1)x^{(1)}x(1) 已经满足精度要求，则可以停止计算，否则需要修正。记

Δx≜x∗−x(1)\Delta x \triangleq x_* - x^{(1)} Δx≜x∗​−x(1)

则 Δx\Delta xΔx 满足方程

AΔx=b−Ax(1)A\Delta x = b - Ax^{(1)} AΔx=b−Ax(1)

只要求出 Δx\Delta xΔx，那么与近似解 x(1)x^{(1)}x(1) 相加即可得到准确解。然而直接求解比较困难，因此我们还是求解近似方程

GΔx=b−Ax(1)G\Delta x = b - Ax^{(1)} GΔx=b−Ax(1)

得到一个近似的修正量

Δ~x=G−1(b−Ax(1))\tilde{\Delta}x = G^{-1}(b - Ax^{(1)}) Δ~x=G−1(b−Ax(1))

于是修正后的近似解为

x(2)=x(1)+Δ~x=x(1)+M−1(b−Ax(1))x^{(2)} = x^{(1)} + \tilde{\Delta}x = x^{(1)} + M^{-1}(b - Ax^{(1)}) x(2)=x(1)+Δ~x=x(1)+M−1(b−Ax(1))

如果 x(2)x^{(2)}x(2) 已经满足精度要求，则停止计算，否则继续按以上的方式进行修正。不断重复以上步骤，于是，我们就得到一个向量序列

x(1), x(2), …, x(k), …x^{(1)}, \, x^{(2)}, \, \ldots, \, x^{(k)}, \, \ldots x(1),x(2),…,x(k),…

它们都是真解 x∗x_*x∗​ 的近似值，且满足下面的递推关系

x(k+1)=x(k)+G−1(b−Ax(k))k=1,2,…x^{(k+1)} = x^{(k)} + G^{-1}(b - Ax^{(k)}) \quad k = 1, 2, \ldots x(k+1)=x(k)+G−1(b−Ax(k))k=1,2,…

这就构成了一个迭代方法。由于每次迭代的格式是一样的，因此称为定常迭代法。现在回过头来看对于系数矩阵 AAA 的近似矩阵 GGG，首先它作为直接进行求解的矩阵，一定要好算。其次，要使得迭代序列尽可能快地收敛。

关于 GGG 的选择，目前比较常用的是矩阵分裂。称

A=M−NA = M - N A=M−N

为 AAA 的一个矩阵分裂，其中 MMM 非奇异。给定一个矩阵分裂，则原方程组 Ax=bAx = bAx=b 就等价于

Mx=Nx+b.Mx = Nx + b. Mx=Nx+b.

于是我们就可以构造出以下的迭代格式

Mx(k+1)=Nx(k)+bk=0,1,…,Mx^{(k+1)} = Nx^{(k)} + b \quad k = 0, 1, \dots, Mx(k+1)=Nx(k)+bk=0,1,…,

或

x(k+1)=M−1Nx(k)+M−1b≜Gx(k)+fk=0,1,…,x^{(k+1)} = M^{-1}Nx^{(k)} + M^{-1}b \triangleq Gx^{(k)} + f \quad k = 0, 1, \dots, x(k+1)=M−1Nx(k)+M−1b≜Gx(k)+fk=0,1,…,

其中 G≜M−1NG \triangleq M^{-1}NG≜M−1N 称为迭代矩阵。

3. 代码实现

要求输入矩阵 AAA 以及 bbb。为了检查 AAA 的非奇异性，考虑使用 LU 分解。若得到的上三角矩阵 UUU 的对角线上含有 000，则报错。随后对 AAA 选取对角线主元，进行分解。此外，还需要手动设定一个可容忍误差，一般选取为

∥b−Ax(k)∥∥b−Ax(0)∥<tol\frac{\|b - Ax^{(k)}\|}{\|b - Ax^{(0)}\|} < \text{tol} ∥b−Ax(0)∥∥b−Ax(k)∥​<tol

其中 tol\text{tol}tol 为手动选择，一般可取为 10−810^{-8}10−8。

这个形式我是第一次见，不过其并不可靠。譬如假使一开始选取的 x(0)x^{(0)}x(0) 使得残差数量级达到惊人的 101010^{10}1010，那么最后迭代将停留在 ∥b−Ax(k)∥<1010tol\|b - Ax^{(k)}\| < 10^{10} \text{tol}∥b−Ax(k)∥<1010tol，这几乎是不可接受的。因此还需考虑另一个判据

∥b−Ax(k)∥<tol\|b - Ax^{(k)}\| < \text{tol} ∥b−Ax(k)∥<tol

具体写程序时，多加上的判据并没有新的计算，不过是多了一次比较大小。
