最小二乘法

1. 问题

在实际科学实验或工程中，我们获得的数据往往包含“噪声”或误差。最小二乘法的目的是：

  • 参数估计：在已知模型形式（如直线、曲线）的情况下，估算出最符合数据特征的参数。
  • 函数拟合：找到一条能够最佳拟合散点图的曲线。
  • 预测：利用拟合出的模型，对未来的数据进行预测。

所谓二乘，即为平方的意思。然而关于这个词的演化，大多数地方都未提及。目前根据 deepseek 的考证，先是日本在翻译英文时，采用了中国典籍中表示两个数相乘的二乘，随后中国又反过来用日语中的二乘。然而它给出的网站我打不开，因此难辨真假。

这里以最简单的线性回归为例：假设我们有一组观测数据 (x1,y1), (x2,y2), …, (xn,yn)(x_1, y_1), \, (x_2, y_2), \, \dots, \, (x_n, y_n)(x1​,y1​),(x2​,y2​),…,(xn​,yn​) ，我们想找一条直线来拟合这些点

在处理多个自变量时，我们从具体地情形入手：先看一个具体的测量点，譬如比如样本 iii ，测量了结果为 yiy_iyi​ ，譬如体重；以及一组因素： xi1,xi2,…,xikx_{i1}, x_{i2}, \dots, x_{ik}xi1​,xi2​,…,xik​ ，譬如身高、体脂、年龄。我们期望

2. 最值问题

线性最小二乘问题就是求解下面的最值问题：

min⁡x∈Rn∥Ax−b∥22\min_{x \in \mathbb{R}^n} \|Ax - b\|_2^2 x∈Rnmin​∥Ax−b∥22​

其中 A∈Rm×n,b∈RmA \in \mathbb{R}^{m \times n}, b \in \mathbb{R}^mA∈Rm×n,b∈Rm。

这句话很费解，而且是有问题的。我们将之与前面关于线性回归的讨论对应起来。这里的 AAA 就是我们测量的一组因素，它的行代表一个个样本，列代表不同的因素。在前面，我们指出，为了配合常数项，其第一列需要全为 111。而 bbb 为测量的结果，xxx 则是相关的系数。那么这个表达式就是选取合适的 xxx 使残差最小。当然，有时候 AAA 的第一列并非全为 111，这是因为我们对其进行了一步预处理，即在方程

yi=β0+β1xi1+β2xi2+⋯+ϵiy_i = \beta_0 + \beta_1 x_{i1} + \beta_2 x_{i2} + \dots + \epsilon_i yi​=β0​+β1​xi1​+β2​xi2​+⋯+ϵi​

通过对常数项 β0\beta_0β0​ 的求导得到

β0=yˉ−∑j=1nβjxˉj\beta_0 = \bar{y} - \sum_{j=1}^n \beta_j \bar{x}_j β0​=yˉ​−j=1∑n​βj​xˉj​

回代可得

(yi−yˉ)=β1(xi1−xˉ1)+β2(xi2−xˉ2)+⋯+ϵi(y_i - \bar{y}) = \beta_1 (x_{i1} - \bar{x}_1) + \beta_2 (x_{i2} - \bar{x}_2) + \dots + \epsilon_i (yi​−yˉ​)=β1​(xi1​−xˉ1​)+β2​(xi2​−xˉ2​)+⋯+ϵi​

于是令 yi∗=yi−yˉy_i^* = y_i - \bar{y}yi∗​=yi​−yˉ​，以及 xij∗=xij−xˉjx_{ij}^* = x_{ij} - \bar{x}_jxij∗​=xij​−xˉj​ 可得

yi∗=β1xi1∗+β2xi2∗+⋯+ϵiy_i^* = \beta_1 x_{i1}^* + \beta_2 x_{i2}^* + \dots + \epsilon_i yi∗​=β1​xi1∗​+β2​xi2∗​+⋯+ϵi​

现在就没有常数项了。从计算资源上看，这样的变量替换多进行了 ∼mn\sim mn∼mn 次减法浮点数计算，而由此简化后的矩阵节省了 ∼2mn\sim 2mn∼2mn 次乘法浮点数计算。考虑到乘法和减法实现的难度差别，因此是有利可图的。

如果 m=nm = nm=n 且 AAA 非奇异，那这就是线性方程组的问题，找出满足 Ax=bAx = bAx=b 的 xxx 。

如果 m>nm > nm>n ，方程组数量大于未知数数量，这样的方程组通常无解，其具体情况在前面的博客中进行了梳理。不过我们可以考虑使得

J(x)≜∥Ax−b∥22J(x) \triangleq \|Ax - b\|_2^2 J(x)≜∥Ax−b∥22​

取最小值的 xxx 。可以证明，J(x)J(x)J(x) 是关于 xxx 的二次函数，而且是凸函数。考虑求导

∂J∂x=0  ⟺  ATAx−ATb=0\frac{\partial J}{\partial x} = 0 \iff A^T A x - A^T b = 0 ∂x∂J​=0⟺ATAx−ATb=0

这表明，使得 J(x)J(x)J(x) 最小的解 x∗x_*x∗​ 是方程

ATAx−ATb=0A^T A x - A^T b = 0 ATAx−ATb=0

的真实解。现在考虑如果 AAA 不满秩，即 rank⁡A<n\operatorname{rank}A < nrankA<n ，那么根据定理

rank(ATA)=rank(A)\text{rank}(A^T A) = \text{rank}(A) rank(ATA)=rank(A)

此时 ATAA^T AATA 奇异。那么现在，存在着无解和无穷多解两种情形。简单地说，二次凸的函数一定有最小值，所以必然存在一个解。不过还是可以说明一下，这种样式的方程存在解。需要利用已知结论： ATAA^T AATA 的列空间和 ATA^TAT 相同，即

C(ATA)=C(AT)\mathcal{C}(A^T A) = \mathcal{C}(A^T) C(ATA)=C(AT)

那么 ATbA^T bATb 就一定在 ATAA^T AATA 的列空间，于是方程 (ATA)x=ATb(A^T A) x = A^T b(ATA)x=ATb 一定有解。

如果 m<nm<nm<n，这里假定 AAA 满秩，那么存在无穷多解。这时我们通常寻求最小范数解，于是原问题就转化为下面的约束优化问题

min⁡Ax=b12∥x∥22\min_{Ax=b} \frac{1}{2} \|x\|_2^2 Ax=bmin​21​∥x∥22​

对应的 Lagrange 函数为

L(x,λ)=12∥x∥22+λT(Ax−b)\mathcal{L}(x, \lambda) = \frac{1}{2} \|x\|_2^2 + \lambda^T (Ax - b) L(x,λ)=21​∥x∥22​+λT(Ax−b)

其中 λ=[λ1,…,λm]T\lambda = [\lambda_1, \ldots, \lambda_m]^Tλ=[λ1​,…,λm​]T 是 Lagrange 乘子。问题的解就是 L(x,λ)\mathcal{L}(x, \lambda)L(x,λ) 的鞍点，即：

∂L∂x=x+ATλ=0\frac{\partial \mathcal{L}}{\partial x} = x + A^T \lambda = 0 ∂x∂L​=x+ATλ=0

写成矩阵形式为

[IATA0][xλ]=[0b]\begin{bmatrix} I & A^T \\ A & 0 \end{bmatrix} \begin{bmatrix} x \\ \lambda \end{bmatrix} = \begin{bmatrix} 0 \\ b \end{bmatrix} [IA​AT0​][xλ​]=[0b​]

如果能解出 λ\lambdaλ ，那么就能够得到 xxx 。得到有关 λ\lambdaλ 的方程为

−(AAT)λ=b-(A A^T) \lambda = b −(AAT)λ=b

如果 AAA 满秩，那么 AATA A^TAAT 非奇异，故而存在唯一解。

通过前面的讨论可知，只要 AAA 满秩，就一定能够找到满足要求的唯一解。因此这里不会考虑不满秩的情形，因为 ppt 上也没说怎么回事。

所以对于超定方程组，我们需要的解为

min⁡x∈Rn∥Ax−b∥22  ⟹  x=(ATA)−1ATb\min_{x \in \mathbb{R}^n} \|Ax - b\|_2^2 \implies x = (A^T A)^{-1}A^T b x∈Rnmin​∥Ax−b∥22​⟹x=(ATA)−1ATb

注意，由于 AAA 不是方阵，因此不能进行化简。

对于欠定方程组

min⁡Ax=b12∥x∥22  ⟹  x=AT(AAT)−1b\min_{Ax=b} \frac{1}{2} \|x\|_2^2 \implies x = A^T (A A^T)^{-1} b Ax=bmin​21​∥x∥22​⟹x=AT(AAT)−1b

然而，我们不能直接按照解析形式进行计算，因为这样会带来无穷无尽的误差。在前面，我们介绍了非奇异矩阵的条件数，现在将之推广至非方阵上。

ATAA^TAATA 作为方阵，若其非奇异，则特征值为 λi\lambda_iλi​ ，可据此定义 AAA 的奇异值为

σi=λi\sigma_i = \sqrt{\lambda_i} σi​=λi​​

于是条件数为

κ(A)=∣σ∣max⁡∣σ∣min⁡\kappa(A) = \frac{|\sigma|_{\max}}{|\sigma|_{\min}} κ(A)=∣σ∣min​∣σ∣max​​

这里需要说明的是，ATAA^TAATA 和 AATAA^TAAT 的非零特征值完全一致，其余特征值全为 000。如果是超定方程组，也就是 m>nm > nm>n ，那么选择 ATAA^TAATA 这个方阵维度更小。当然，如果是欠定方程组，自然要选取 AATAA^TAAT 。此外， ATAA^TAATA 和 AATAA^TAAT 都是半正定的，考虑如下的证明：

(ATA)v=λv(A^T A) v = \lambda v (ATA)v=λv

在等式左右两边同时左乘 vTv^TvT ，并经过一番化简可得

∥Av∥22=λ∥v∥22  ⟹  λ=∥Av∥22∥v∥22≥0\|Av\|_2^2 = \lambda \|v\|_2^2 \implies \lambda = \frac{\|Av\|_2^2}{\|v\|_2^2} \ge 0 ∥Av∥22​=λ∥v∥22​⟹λ=∥v∥22​∥Av∥22​​≥0

所以当 AAA 满秩时，上式不取等。因此在我们的问题中，可以将不等式去掉。这时候

κ(ATA)=σmax⁡2σmin⁡2\kappa(A^TA) = \frac{\sigma_{\max}^2}{\sigma_{\min}^2} κ(ATA)=σmin2​σmax2​​

条件数急速放大。

3. QR 分解

考虑方程

min⁡∥Ax−b∥2\min \|Ax - b\|_2 min∥Ax−b∥2​

我们希望能够将之进行化简，但是不改变条件数，这使我们能够想到正交矩阵 QQQ

min⁡∥Ax−b∥2=min⁡∥Q(Ax−b)∥2=min⁡∥QAx−Qb∥2\min \|Ax - b\|_2 = \min \|Q(Ax - b)\|_2 = \min \|Q Ax - Q b\|_2 min∥Ax−b∥2​=min∥Q(Ax−b)∥2​=min∥QAx−Qb∥2​

注意，这里要求 Q∈RmQ \in \mathbb{R}^mQ∈Rm 。现在不妨回代一下看看

x=(ATQTQA)−1ATQTQbx = (A^TQ^TQ A)^{-1}A^TQ^TQ b x=(ATQTQA)−1ATQTQb

这是很平凡的，在里面插入单位矩阵，不过类似的技巧在量子力学、量子光学中很有用。如果我们定义一个新的矩阵 R≜QA∈Rm×nR \triangleq Q A \in \mathbb{R}^{m \times n}R≜QA∈Rm×n ，那么公式就变成了：

x=(RTR)−1RT(Qb)x = (R^T R)^{-1} R^T (Q b) x=(RTR)−1RT(Qb)

现在，最关键的问题来了：这个 QQQ 应该怎么选，这个“简化”才有意义？如果 RRR 是方阵，那么

x=(RTR)−1RT(Qb)=R−1Qbx = (R^T R)^{-1} R^T (Q b) = R^{-1} Qb x=(RTR)−1RT(Qb)=R−1Qb

还是差点意思，因此如果 RRR 是三角矩阵，那就好了，因为可以用追赶法求解。如果 RRR 具有如下的形式

R=[R10]R = \begin{bmatrix} R_1 \\ 0 \end{bmatrix} R=[R1​0​]

RTR=[R1T0][R10]=R1TR1R^T R = \begin{bmatrix} R_1^T & 0 \end{bmatrix} \begin{bmatrix} R_1 \\ 0 \end{bmatrix} = R_1^T R_1 RTR=[R1T​​0​][R1​0​]=R1T​R1​

现在考虑后半部分

RT(Qb)=[R1T0]Qb=[R1T0][c1c2]=R1Tc1R^T (Qb) = \begin{bmatrix} R_1^T & 0 \end{bmatrix} Qb = \begin{bmatrix} R_1^T & 0 \end{bmatrix} \begin{bmatrix} c_1 \\ c_2 \end{bmatrix} = R_1^T c_1 RT(Qb)=[R1T​​0​]Qb=[R1T​​0​][c1​c2​​]=R1T​c1​

如果 R1R_1R1​ 可逆，公式立刻简化为

x=R1−1(R1T)−1R1Tc1  ⟹  x=R1−1c1x = R_1^{-1} (R_1^T)^{-1} R_1^T c_1 \implies x = R_1^{-1} c_1 x=R1−1​(R1T​)−1R1T​c1​⟹x=R1−1​c1​

也就是说，我们希望实现

R=QA  ⟹  A=Q−1R=QTRR = QA \implies A = Q^{-1} R = Q^T R R=QA⟹A=Q−1R=QTR

其中 QQQ 为 mmm 阶正交矩阵，RRR 为一种特殊的三角矩阵。

另一方面，可以构造 QTQ=ImQ^TQ = I_mQTQ=Im​ ，但是 Q∈Rn×mQ \in \mathbb{R}^{n \times m}Q∈Rn×m 。此时 R=QA∈RnR = QA \in \mathbb{R}^{n}R=QA∈Rn 。自然，我们要求 RRR 为三角矩阵。
