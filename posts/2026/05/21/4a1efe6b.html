预处理共轭梯度法

1. 问题

在前面，我们已经知道了共轭梯度法满足

∥x(k)−x∗∥A∥x(0)−x∗∥A≤2(κ−1κ+1)k\frac{\|x^{(k)} - x_*\|_A}{\|x^{(0)} - x_*\|_A} \leq 2 \left( \frac{\sqrt{\kappa} - 1}{\sqrt{\kappa} + 1} \right)^k ∥x(0)−x∗​∥A​∥x(k)−x∗​∥A​​≤2(κ​+1κ​−1​)k

其中 κ\kappaκ 为对称矩阵 AAA 的谱条件数，其具体表达式为

κ=∣λ∣max⁡∣λ∣min⁡\kappa = \frac{|\lambda|_{\max}}{|\lambda|_{\min}} κ=∣λ∣min​∣λ∣max​​

其值越大，共轭梯度法的收敛速度越慢，尤其是处理一些巨大矩阵时。因此我们希望通过一个对称正定矩阵 MMM 对原系统进行变换，使得变换后的系统特征值更加集中。将预处理矩阵 MMM 分解为 M=EETM = EE^TM=EET，那么有

(E−1AE−T)(ETx)=E−1b(E^{-1}AE^{-T})(E^T x) = E^{-1}b (E−1AE−T)(ETx)=E−1b

令新矩阵 A~=E−1AE−T\tilde{A} = E^{-1}AE^{-T}A~=E−1AE−T ，新变量 x~=ETx\tilde{x} = E^T xx~=ETx ，新常数项 b~=E−1b\tilde{b} = E^{-1}bb~=E−1b 。现在的方程变成了求解

A~x~=b~\tilde{A}\tilde{x} = \tilde{b} A~x~=b~

理想的情况是

M=A  ⟹  A~=E−1AE−T=E−1EETE−T=InM = A \implies \tilde{A} = E^{-1}AE^{-T} = E^{-1}EE^T E^{-T} = I_n M=A⟹A~=E−1AE−T=E−1EETE−T=In​

此时 κ=1\kappa = 1κ=1，带入前面的收敛条件得到

∥x(k)−x∗∥A∥x(0)−x∗∥A≤0\frac{\|x^{(k)} - x_*\|_A}{\|x^{(0)} - x_*\|_A} \leq 0 ∥x(0)−x∗​∥A​∥x(k)−x∗​∥A​​≤0

表明一步即可。然而我们解出的方程为

x~=b~  ⟹  x=E−TE−1b=(EET)−1b=A−1b\tilde{x} = \tilde{b} \implies x = E^{-T}E^{-1} b = (EE^T)^{-1} b = A^{-1} b x~=b~⟹x=E−TE−1b=(EET)−1b=A−1b

这并未改善问题。因此，现在的问题在于如何选择合适的 MMM。

2. 数学原理

这里考虑一种简单的方法：取 MMM 为 AAA 的对角线元素。这里我们介绍一个正定矩阵的性质，前面应该是没有的：

正定矩阵 AAA 的对角线元素全部大于 000。考虑二次型

eiTAei=eiT(a1ia2i⋮ani)=aii>0e_i^T A e_i = e_i^T \begin{pmatrix} a_{1i} \\ a_{2i} \\ \vdots \\ a_{ni} \end{pmatrix} = a_{ii} > 0 eiT​Aei​=eiT​​a1i​a2i​⋮ani​​​=aii​>0

所以能够写出

E=(a110000a2200⋮⋮⋱⋮000ann)E = \begin{pmatrix} \sqrt{a_{11}} & 0 & 0 & 0 \\ 0 & \sqrt{a_{22}} & 0 & 0 \\ \vdots & \vdots & \ddots & \vdots \\ 0 & 0 & 0 & \sqrt{a_{nn}} \end{pmatrix} E=​a11​​0⋮0​0a22​​⋮0​00⋱0​00⋮ann​​​​

那么

A~ij=E−1AE−T=Eil−1AlkEkj−T=1allδilalk1akkδkj=aijaiiajj\tilde{A}_{ij} = E^{-1}AE^{-T} = E^{-1}_{il} A_{lk} E^{-T}_{kj} = \frac{1}{\sqrt{a_{ll}}} \delta_{il} a_{lk} \frac{1}{\sqrt{a_{kk}}} \delta_{kj} = \frac{a_{ij}}{\sqrt{a_{ii}a_{jj}}} A~ij​=E−1AE−T=Eil−1​Alk​Ekj−T​=all​​1​δil​alk​akk​​1​δkj​=aii​ajj​​aij​​

这里的求和稍微有些问题，不过不管了。那么这个矩阵有两个特点

  • 迹

此外，考虑将 Gershgorin 圆盘定理应用在 A~\tilde{A}A~ 上：

A~\tilde{A}A~ 的所有特征值都落在复平面上的 nnn 个圆盘之内，这些圆盘以 ci=a~ii=1c_i = \tilde{a}_{ii} = 1ci​=a~ii​=1 为圆心，Ri=∑j≠i∣a~ij∣=∑j≠i∣aij∣aiiajjR_i = \sum_{j \neq i} |\tilde{a}_{ij}| = \sum_{j \neq i} \frac{|a_{ij}|}{\sqrt{a_{ii} a_{jj}}}Ri​=∑j=i​∣a~ij​∣=∑j=i​aii​ajj​​∣aij​∣​ 为半径。

这表明，如果矩阵 AAA 具有对角优势，那么半径 RiR_iRi​ 会比较小，至少是小于 111 的，所有的特征值都被关在了以 111 为中心、半径较小的圆盘簇里。相比原矩阵，特征值自然大大向 111 集中了。

这里我调用了昨天编写的 CG 函数，并将对于 AAA 对称正定性的判断留在了 CG 函数中，这里需要说明：

根据 Sylvester 惯性定律，合同变换不改变矩阵特征值的正、零和负的数量。因此，如果 A~\tilde{A}A~ 正定，那么其特征值全为正值，从而 AAA 的特征值为正值，也就是 AAA 正定。至于对称性，还是比较显然的。

另一方面，对于正定性的判断，AI 说可以留在 pTApp^TAppTAp 中进行而不必去调用 chol 函数。这样虽然节省了大量的计算资源，但是并不能确保 AAA 的正定性，所以我就不进行修改了。

对于 5×55 \times 55×5 的对称正定矩阵，其迭代了 555 次，而对于 1000×10001000 \times 10001000×1000 的随机对称正定矩阵，反而出现了报错
