Cholesky 分解

1. 问题

最初来自一名法国陆军测绘局的炮兵司令官、大地测量学家和数学家 André-Louis Cholesky，他遇到了一些超定方程组

{a1x1+a2x2+⋯+anxn+K1=0b1x1+b2x2+⋯+bnxn+K2=0⋮l1x1+l2x2+⋯+lnxn+Kp=0\begin{cases} a_1 x_1 + a_2 x_2 + \dots + a_n x_n + K_1 = 0 \\ b_1 x_1 + b_2 x_2 + \dots + b_n x_n + K_2 = 0 \\ \vdots \\ l_1 x_1 + l_2 x_2 + \dots + l_n x_n + K_p = 0 \end{cases} ⎩⎨⎧​a1​x1​+a2​x2​+⋯+an​xn​+K1​=0b1​x1​+b2​x2​+⋯+bn​xn​+K2​=0⋮l1​x1​+l2​x2​+⋯+ln​xn​+Kp​=0​

其中 n>pn>pn>p，并且带有附加条件

F=x12+x22+⋯+xn2F = x_1^2 + x_2^2 + \dots + x_n^2 F=x12​+x22​+⋯+xn2​

取最小值。Lagrange 给出了一种方法，构造 Lagrange 函数

F=F−2λ1f1−2λ2f2−⋯−2λpfp\mathfrak{F} = F - 2\lambda_1 f_1 - 2\lambda_2 f_2 - \dots - 2\lambda_p f_p F=F−2λ1​f1​−2λ2​f2​−⋯−2λp​fp​

其中 fff 为相应的约束。令偏导数为 000

∂F∂xj=0  ⟹  xj=ajλ1+bjλ2+⋯+ljλp\frac{\partial \mathfrak{F}}{\partial x_j} = 0 \quad \implies \quad x_j = a_j \lambda_1 + b_j \lambda_2 + \dots + l_j \lambda_p ∂xj​∂F​=0⟹xj​=aj​λ1​+bj​λ2​+⋯+lj​λp​

也即是说，我们将 nnn 个未知数 xix_ixi​ 的求解转化为了 ppp 个参数 λi\lambda_iλi​ 的求解。将之回代入线性方程组，得到

{A11λ1+A12λ2+⋯+A1pλp+K1=0A12λ1+A22λ2+⋯+A2pλp+K2=0⋮A1pλ1+A2pλ2+⋯+Appλp+Kp=0 \begin{cases} A_{11} \lambda_1 + A_{12} \lambda_2 + \dots + A_{1p} \lambda_p + K_1 = 0 \\ A_{12} \lambda_1 + A_{22} \lambda_2 + \dots + A_{2p} \lambda_p + K_2 = 0 \\ \vdots \\ A_{1p} \lambda_1 + A_{2p} \lambda_2 + \dots + A_{pp} \lambda_p + K_p = 0 \end{cases} ⎩⎨⎧​A11​λ1​+A12​λ2​+⋯+A1p​λp​+K1​=0A12​λ1​+A22​λ2​+⋯+A2p​λp​+K2​=0⋮A1p​λ1​+A2p​λ2​+⋯+App​λp​+Kp​=0​

其中 A11=∑aj2, A12=∑ajbjA_{11} = \sum a_j^2,\ A_{12} = \sum a_j b_jA11​=∑aj2​, A12​=∑aj​bj​ 等。让我们考虑更一般的问题

{a11λ1+a12λ2+⋯+a1pλp+K1=0a12λ1+a22λ2+⋯+a2pλp+K2=0…a1pλ1+a2pλ2+⋯+appλp+Kp=0\begin{cases} a_{11}\lambda_1 + a_{12}\lambda_2 + \dots + a_{1p}\lambda_p + K_1 = 0 \\ a_{12}\lambda_1 + a_{22}\lambda_2 + \dots + a_{2p}\lambda_p + K_2 = 0 \\ \dots \\ a_{1p}\lambda_1 + a_{2p}\lambda_2 + \dots + a_{pp}\lambda_p + K_p = 0 \end{cases} ⎩⎨⎧​a11​λ1​+a12​λ2​+⋯+a1p​λp​+K1​=0a12​λ1​+a22​λ2​+⋯+a2p​λp​+K2​=0…a1p​λ1​+a2p​λ2​+⋯+app​λp​+Kp​=0​

这里对角线上的 a11, a22a_{11}, \, a_{22}a11​,a22​ 等元素均为正值。将其写为矩阵形式

Aλ+K=0A\lambda + K = 0 Aλ+K=0

注意，这里并未要求系数矩阵正定。为了解这个方程，当时已经知道 Gauss 消去法。然而在没有电子计算机的时代，还是太慢了。Cholesky 注意到系数矩阵的特性，于是考虑一个辅助的线性方程组

{α11y1+H1=0α21y1+α22y2+H2=0α31y1+α32y2+α33y3+H3=0⋮αp1y1+αp2y2+αp3y3+⋯+αppyp+Hp=0\begin{cases} \alpha_{11}y_1 + H_1 = 0 \\ \alpha_{21}y_1 + \alpha_{22}y_2 + H_2 = 0 \\ \alpha_{31}y_1 + \alpha_{32}y_2 + \alpha_{33}y_3 + H_3 = 0 \\ \vdots \\ \alpha_{p1}y_1 + \alpha_{p2}y_2 + \alpha_{p3}y_3 + \dots + \alpha_{pp}y_p + H_p = 0 \end{cases} ⎩⎨⎧​α11​y1​+H1​=0α21​y1​+α22​y2​+H2​=0α31​y1​+α32​y2​+α33​y3​+H3​=0⋮αp1​y1​+αp2​y2​+αp3​y3​+⋯+αpp​yp​+Hp​=0​

也就是

Ly+H=0Ly + H = 0 Ly+H=0

G=G−2κ1g1−2κ2g2−⋯−2κpgp\mathfrak{G} = G - 2\kappa_1 g_1 - 2\kappa_2 g_2 - \cdots - 2\kappa_p g_p G=G−2κ1​g1​−2κ2​g2​−⋯−2κp​gp​

∂G∂yi=2yi−2∑k=ipκk⋅αki=0\frac{\partial \mathfrak{G}}{\partial y_i} = 2y_i - 2\sum_{k=i}^p \kappa_k \cdot \alpha_{ki} = 0 ∂yi​∂G​=2yi​−2k=i∑p​κk​⋅αki​=0

yi=∑k=ipαkiκk y_i = \sum_{k=i}^p \alpha_{ki} \kappa_k yi​=k=i∑p​αki​κk​

2. 数学原理

对于 Cholesky 分解，有如下定理： A∈Rn×nA \in \mathbb{R}^{n \times n}A∈Rn×n 对称正定，则存在唯一的对角线元素全为正的下三角矩阵 LLL，使得

A=LLTA = L L^T A=LLT

现在我们推导出 AAA 的正定性。考虑对于任意非零向量 vvv

vTAv=vT(LLT)v=(LTv)T(LTv)=∥LTv∥2≥0v^T A v = v^T (LL^T) v = (L^T v)^T (L^T v) = \|L^T v\|^2 \ge 0 vTAv=vT(LLT)v=(LTv)T(LTv)=∥LTv∥2≥0

这表明 AAA 至少是一个半正定矩阵，即使 AAA 或者 vvv 可能包含复数元素。不过鉴于计算软件在复数处理上的资源消耗，现在仅考虑实数，也就是 A∈Rn×nA \in \mathbb{R}^{n \times n}A∈Rn×n。此外，分解得到的 LLL 也要如此。那么

det⁡L=∏i=1pαii>0\det L = \prod_{i = 1}^p \alpha_{ii} > 0 detL=i=1∏p​αii​>0

这是因为 αii\alpha_{ii}αii​ 作为实数开根号的结果，并且让还会处于分母上，所以必定大于 000。因此 LLL 为可逆矩阵，从而 LTv≠0L^T v \neq 0LTv=0，因此 AAA 是正定的。

现在给出一个有关的定理：

αii2=det⁡(Ai)det⁡(Ai−1)\alpha_{ii}^2 = \frac{\det(A_i)}{\det(A_{i-1})} αii2​=det(Ai−1​)det(Ai​)​

其中 AiA_iAi​ 为 AAA 的 iii 阶主子式。我们不去证明此公式，不过可以看出，AAA 的所有顺序主子式必须大于 000，这也正是 AAA 正定的充要条件。
